常用的linux命令

添加IPv6地址

$ sudo ip -6 addr add IP6地址 dev 网口号
例如： sudo ip -6 addr add 2001:0db8:0:f101::1/64 dev wlp2s0

测试IPv6地址

本地网络不通过路由器采用

ping6 -I 网口号　IP地址(不需要加后面的掩码位数)

测试外网使用

ping6 IP地址(不需要加后面的掩码位数)

添加到主机的路由

$ route add -host 192.168.1.2 dev eth0
$ route add -host 10.20.30.148 gw 10.20.30.40 #添加到10.20.30.148的网关

添加到网络的路由

$ route add -net 10.20.30.40 netmask 255.255.255.248 eth0 #添加10.20.30.40的网络
$ route add -net 10.20.30.48 netmask 255.255.255.248 gw 10.20.30.41 #添加10.20.30.48的网络
$ route add -net 192.168.1.0/24 eth1

添加默认路由

$ route add default gw 192.168.1.1

删除路由

$ route del -host 192.168.1.2 dev eth0:0
$ route del -host 10.20.30.148 gw 10.20.30.40
$ route del -net 10.20.30.40 netmask 255.255.255.248 eth0
$ route del -net 10.20.30.48 netmask 255.255.255.248 gw 10.20.30.41
$ route del -net 192.168.1.0/24 eth1
$ route del default gw 192.168.1.1

IPv6地址分类

本地链路地址(link)

IPv6本地链路地址的作用在于发现邻居节点和在无状态自动配置中链路本地上节点之间的通信。并且只能在连接到同一本地链路的节点之间通讯，不能跨路由器进行通讯和路由转发.

公网地址(globe)

可以被外网访问，能够通过路由进行通讯和路由器转发.
